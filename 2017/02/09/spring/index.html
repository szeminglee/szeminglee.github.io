Spring

当春风拂面那一刻，我就无法回头，回头看那严冬经历的种种

或许人生不会有太多的交集，但我依旧在心里藏了一块“雪地”

那样当我心中充满火热，还可以找寻一丝凉意

我曾经无数次的问自己，究竟什么时候才能成熟

但是，当自己发现自己成熟那一刻开始

却越来越变得孤独

或许在别人看来，这是一种病态

我本以为我会跟他会再有交集

但是现实总是那么的不留情面

从2009年开始的暗恋

到后面的明恋

我从来都不觉得早恋是一种罪

呵呵！

你知道那时候我是多么纯真的笑么

这么多年过去了

生活中

我就觉得生活仿佛充满无尽的希望。
